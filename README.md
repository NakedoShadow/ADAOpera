# ADA Opera — La Voix de la Reine

Browser-native DAW. 32 instruments, DJ Suite, CRYSTALLINE AI denoising, NotebookLM integration, 256 features, 16 languages.

By [Nakedo Corp](https://nakedo.ai) / Shadows Company.

## Features

### CRYSTALLINE — AI Audio Denoising
6-stage pipeline: RNNoise, DeepFilterNet3, MossFormer2, Resemble Enhance, SpeechBrain, Validation.

### VIVALDI — AI Orchestra Conductor
32-instrument orchestra across 4 sections. Real-time AI conducting with emotion mapping.

### DJ Platine
4-skin turntable system: Vinyl, CDJ, Old School, Futuristic.

### Piano & Violon
Discrete + continuous synthesis operators with expression control.

### Spectrum Visualizer
6-mode galaxy: Waveform, FFT, Spectrogram, Phase Scope, Circular, Bars.

### Mixer & Effects
Professional channel mixing with signal processing.

### i18n — 16 Languages
EN, FR, ES, DE, IT, PT, JA, KO, ZH, AR, RU, HI, TR, NL, PL, SV.

## Tech Stack

- **React 19** + **TypeScript 5.8**
- **Tone.js 15** — Web Audio synthesis
- **Zustand 5** — State management
- **Vite 6** — Fast builds (~155KB gzip)
- **Tauri v2** (optional) — Desktop native audio DSP

## Install

```bash
npm install
npm run dev    # Dev server on port 8300
```
